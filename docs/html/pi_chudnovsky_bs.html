pi_chudnovsky_bs module¶

Python3 program to calculate Pi using python long integers, binary splitting and the Chudnovsky algorithm

See: http://www.craig-wood.com/nick/articles/pi-chudnovsky/ for more info

Nick Craig-Wood <nick@craig-wood.com>

pi_chudnovsky_bs.pi_chudnovsky_bs(digits)¶

Compute int(pi * 10**digits)

This is done using Chudnovsky’s series with binary splitting

pi_chudnovsky_bs.sqrt(n, one)¶

Return the square root of n as a fixed point number with the one passed in. It uses a second order Newton-Raphson convgence. This doubles the number of significant figures on each iteration.
